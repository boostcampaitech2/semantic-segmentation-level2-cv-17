## 🛠 Installation
PyPI version:

```$ pip install segmentation-models-pytorch```

Latest version from source:

```$ pip install git+https://github.com/qubvel/segmentation_models.pytorch```

## ⏳ Quick start
### train
``` python3 smp_train.py```
### inference
```python3 smp_inference.py```
